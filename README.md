# Zap Imóveis Dashboard

Dashboard interativo com dados de imóveis para alugar em Fortaleza, coletados automaticamente do Zap Imóveis via web scraping.

##  Sobre o projeto

Este projeto coleta dados de imóveis para aluguel em Fortaleza diretamente do site Zap Imóveis utilizando Selenium, organiza os dados com Pandas e os exibe em um dashboard interativo construído com Streamlit.

## Funcionalidades

- Coleta automática de dados de imóveis (preço, bairro, área, quartos, banheiros e vagas)
- Exportação dos dados para CSV
- Dashboard interativo com filtro por bairro
- Gráfico de quantidade de imóveis por bairro

## Tecnologias utilizadas

- Python 3
- Selenium — automação do navegador para coleta de dados
- Pandas — organização e exportação dos dados
- Streamlit — criação do dashboard interativo
